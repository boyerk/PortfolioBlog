The next project here is my Amazon Alexa Skill. This Alexa skill was made with Amazon Web Services. The skill I created is a quick poker guide. I also took this project further for my final and published it with Amazon so that it can be used off of any Alexa device.


For this project we were asked to view some sort of language or framework and create a basic descriptive, cheat sheet that we could all share with each other if we ever needed to use that language/framework in the future. The framwork I chose was VIM. VIM is a text editor and plug in that uses shortcuts and modes to allow you to code in a very fast and efficient way without even using a mouse.
